Very little use, but thought to public it anywho.
Was my first time playing around with Java to make a mod for a personal server of mine with friends.
Turned out to work pretty decently.

Allows you to restore someone to life by capturing a creatures soul in a bottle and rescurrecting them with it, however long-term impacts of such rescurrection persist until certain criteria are met.

Could certainly be taken further and developed more if someone wanted to.
